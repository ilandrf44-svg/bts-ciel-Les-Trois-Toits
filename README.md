# Projet BTS CIEL
Projet collaboratif electronique connecte.

Le projet Les Trois Toits, réalisé au lycée Livet à Nantes en partenariat avec le domaine viticole Les Trois Toits situé à Vertou, a pour objectif de concevoir et mettre en œuvre un système de mesure de température en extérieur afin d’analyser les phénomènes de gel tardif pouvant impacter le vignoble. Le système repose sur le déploiement de dix sondes autonomes réparties sur une distance d’environ cent mètres, capables de mesurer la température et l’humidité, de transmettre les données par communication LoRa et de fonctionner grâce à une alimentation solaire, avec une identification et une position GPS définies lors de l’installation. Les informations collectées sont envoyées vers une passerelle centrale qui permet l’affichage des températures, l’enregistrement des données sur carte SD et le réglage d’un seuil d’alerte
